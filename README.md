# TETRIS 3D

## Description du projet
L'idée de départ est de créer un jeu de tetris en 3D. Les sept pièces du jeu descendent à intervalle régulier sur le plateau afin de former des lignes se supprimant lorsque toutes les cases de cette ligne sont remplies.

## Instructions de configuration
Ce projet a été réalisé dans un environnement Windows. Il nécessite d'installer la librairie glm, ainsi que les librairies open GL suivantes : GLEW, glfw3.
